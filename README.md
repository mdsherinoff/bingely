# Bingely

Stop wondering when you'll finish a show. Start watching with a plan.

Bingely helps movie lovers, TV fans, and anime watchers create personalized
viewing schedules based on their real availability.

## Tech Stack

- Next.js 14 + TypeScript
- Tailwind CSS
- Framer Motion
- TMDB API
- PostgreSQL + Prisma
- Clerk (auth)
- Vercel (deployment)
